# 🐉 발더스 게이트 3 파티 관리자 (BG3 Raid Manager)

발더스 게이트 3 플레이어들을 위한 파티 관리 및 일정 조율 웹 애플리케이션입니다.
Glassmorphism 디자인을 적용하여 신비롭고 고급스러운 분위기를 연출했습니다.

🔗 **배포 주소:** [https://yangjunhyuk333.github.io/bg3-raid-manager](https://yangjunhyuk333.github.io/bg3-raid-manager)

## ✨ 주요 기능

1.  **파티원 일정 관리 (Raid Scheduler)**
    - 파티원 4명의 스케줄을 확인하고 최적의 레이드 시간을 투표/설정할 수 있습니다.
2.  **실시간 채팅 (Chat Room)**
    - 파티원들과 실시간으로 소통하며 전략을 논의할 수 있습니다.
3.  **세이브 파일 분석 (Save Analyzer)**
    - 세이브 파일을 업로드하여 현재 진행 상황(퀘스트, 아이템 등)을 분석합니다.
4.  **반응형 디자인**
    - PC와 모바일 모두에서 완벽하게 동작하는 사용자 경험을 제공합니다.

## 🛠 기술 스택

- **Frontend:** React, Vite
- **Styling:** CSS Modules, Glassmorphism UI
- **Deployment:** GitHub Pages

## 🚀 실행 방법

이 프로젝트를 로컬에서 실행하려면 다음 명령어를 사용하세요.

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```
